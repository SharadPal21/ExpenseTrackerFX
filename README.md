# 💰 ExpenseTrackerFX

A modern **Expense Tracker Desktop Application** built using **JavaFX and Maven** to help users manage daily expenses efficiently.

---

## 🚀 Overview

ExpenseTrackerFX is a lightweight and user-friendly application designed to:

* Track daily expenses 💸
* Store data locally using SQLite 🗄️
* Provide a clean and interactive UI using JavaFX 🎨

---

## ✨ Features

* ➕ Add and manage expenses
* 📋 View all transactions
* 💾 Persistent storage (SQLite database)
* 🧾 Simple and intuitive interface
* ⚡ Fast and lightweight desktop app

---

## 🛠️ Tech Stack

* **Java 21**
* **JavaFX**
* **Maven**
* **SQLite**

## ⚙️ Setup & Installation

### 1️⃣ Clone the repository

```bash id="8mfrkl"
git clone https://github.com/SharadPal21/ExpenseTrackerFX.git
cd ExpenseTrackerFX
```

---

### 2️⃣ Build the project

```bash id="x4h9zt"
mvn clean install
```

---

### 3️⃣ Run the application

```bash id="q81gk7"
java -jar target/ExpenseTrackerFX.jar
```
