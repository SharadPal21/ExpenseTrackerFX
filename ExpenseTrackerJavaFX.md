# Expense Tracker FX

A desktop **Expense Tracker / Personal Finance Manager** built with **JavaFX**, **FXML**, and **SQLite**.  
It helps you record daily expenses, organize them by category, and visualize spending with a pie chart.

## Features

- Add new expenses with date, category, amount, and description
- Delete selected expenses
- View all expenses in a table
- See total expenses and current month spending
- Visualize spending by category with a pie chart
- SQLite database for local persistence
- Modern JavaFX UI with CSS styling

## Tech Stack

- **Java 21**
- **JavaFX 23**
- **FXML**
- **SQLite**
- **Maven**
- **AtlantaFX**
- **ControlsFX**
- **Ikonli**
- **SLF4J / Logback**
